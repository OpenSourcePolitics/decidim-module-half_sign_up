# Decidim::HalfSignup

Enbale half-signup option for users.

## Usage

HalfSignup will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-half_signup"
```

And then execute:

```bash
bundle
```
